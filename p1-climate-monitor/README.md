
# Smart Room Climate Monitor

## Components

- ESP32
- DHT11
- OLED Display
- Buzzer
- Red LED
- Green LED

## Features

- Reads temperature
- Reads humidity
- Displays values on OLED
- Alerts when temperature exceeds 38°C
- Alerts when humidity exceeds 80%

## Libraries

- DHTesp
- Adafruit SSD1306
- Adafruit GFX
