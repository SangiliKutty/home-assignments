# Gas & Fire Safety Alert System

## Project Overview

This project detects gas leaks and fire using an MQ-2 Gas Sensor and Flame Sensor. It provides visual and audible alerts through LEDs and a buzzer to improve home safety.

## Components Used

- Arduino Uno R3
- MQ-2 Gas Sensor
- Flame Sensor
- Active Buzzer
- Red LED
- Yellow LED
- Green LED
- 220Ω Resistors
- Breadboard
- Jumper Wires

## Features

- Detects LPG, smoke, and gas leaks
- Detects fire using the flame sensor
- Three alert levels:
  - Safe
  - Warning
  - Danger
- LED status indication
- Buzzer alarm
- Serial Monitor output

## Wiring

- MQ-2 AOUT → A0
- Flame Sensor DO → D7
- Green LED → D10
- Yellow LED → D11
- Red LED → D12
- Buzzer → D9
